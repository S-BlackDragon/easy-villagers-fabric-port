# Easy Villagers — Unofficial Fabric Port (1.21.1)

> **Unofficial community port** of [Easy Villagers](https://github.com/henkelmax/easy-villagers) by [Max Henkel](https://github.com/henkelmax) for **Fabric 1.21.1**.
>
> All original design and concept credit goes to Max Henkel. If the original author requests removal of this port, it will be taken down immediately.

---

## What is this?

Easy Villagers adds blocks that let you automate and interact with villagers without needing a full village setup. This is a community port to **Fabric 1.21.1** with full feature parity to the original NeoForge mod.

## Blocks

| Block | Description |
|---|---|
| **Trader** | Place a villager and a workstation to trade automatically |
| **Auto Trader** | Automatically trades with the villager using items from input slots |
| **Breeder** | Breeds two villagers to produce a baby villager |
| **Farmer** | Place a villager and food to automatically produce crops |
| **Converter** | Converts a villager into a zombie villager and back to cure it |
| **Incubator** | Grows a baby villager into an adult |
| **Iron Farm** | Place a villager to produce iron ingots and poppies |

## Features

- Capture villagers with the **Villager Catch** item (Shift + Right Click on a villager)
- Release captured villagers by right-clicking on the ground
- Compatible with modded workstations (More Villagers, More Villagers Reemployed, etc.)
- Compatible with hoppers and pipes mods
- Villagers display inside blocks with proper skins and profession hats
- Baby villager support

## Requirements

- Minecraft 1.21.1
- Fabric Loader ≥ 0.15.0
- [Fabric API](https://modrinth.com/mod/fabric-api)

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/)
2. Download [Fabric API](https://modrinth.com/mod/fabric-api)
3. Drop both `.jar` files into your `mods/` folder
